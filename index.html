Memory Game

Watch the growing pattern of lights and sounds, then repeat it back. One new step is added every round — how far can you get?

Mouse, touch, or keyboard — Q W A S

How To Play

The device plays a sequence of colored lights and tones. Repeat the sequence by pressing the same buttons in the same order.

Every round you complete, one new random color is added to the end of the sequence — there's no time limit, so take your time.

Play with mouse, touch, or the keyboard (Q=green, W=red, A=yellow, S=blue).

With Strict Mode on, one mistake ends the run. Turn it off in Settings for a gentler game that just replays the current round after a slip.

Flip on Hard Mode in Settings for a noticeably faster playback speed from round one.
